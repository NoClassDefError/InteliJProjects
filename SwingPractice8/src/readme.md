##绘图程序
###需求
1. 要能够通过鼠标滚轮实现缩放与拖动<br>
2. 要将函数图线组件化，实现点击修改参数或改变显示方案的功能<br>
3. 要能够对图线交点进行计算<br>
4. 能输入任意二元函数并画出图像<br>
5. 能显示坐标栅格<br>
6. 不需要实现框选与手工图像绘制
### 实现方法
####关于缩放与点的坐标表示
1. 各组件的位置与大小应按照窗口大小进行实时运算，但组件与窗口本身应当具有标准的尺寸
2. 绘制点的密度要根据组件的`显示大小`确定
3. 我们将标准尺寸的窗口称为`虚拟屏`,虚拟屏应当将原点作为唯一参考点，拥有标准点密度
####关于函数图像组件化
1. 组件继承`JButton`类并对其边界与显示进行特殊处理，以被选中
2. 组件的显示位置必须要求精确，禁用窗口管理器并以图像原点为参考点
3. 组件新建时输入其函数表达式，调整其颜色
